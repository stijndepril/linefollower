# Instructable

Een instructable is een stappenplan - zonder verdere uitleg - hoe je vertrekkend van de bill of materials en gebruik makend van de technische tekeningen de robot kan nabouwen. Ook de nodige stappen om de microcontroller te compileren en te uploaden staan beschreven.  

### stap 1
bestel alle componenten uit de bill of materials  
### stap 2
pak een stuk stevig wegwerpmateriaal en versnij het tot een grote van 120mm op 100mm
### stap 3
soldeer alle componenten die moeten gesoldeerd worden
### stap 4
bouw het elektrisch op volgens het elektronische schema dat ook te vinden is op deze github
### stap 5
als alles goed is opgebouwd dan kan je nu de code in de microcontroller steken en al eens vlug testen door je vinger op de sensor te houden. de ode is weer te vinden op deze github
### stap 6
als alles goed werkt dan kan je je eerder gemaakte basisplaat erbij nemen en kan je beginnen proberen om alles erop te monteren. probeer juist een afstand van tussen de 9cm en 10cm te houden tussen de sensor en de wielen.
### stap 7
als alles goed staat op de basisplaat dan kan je alles beginnen vastzetten. ik heb dit gedaan met dubbelzijdige tape maar je kan hier veel verschillende dingen voor gebruiken.
### stap 8
normaalgezein als alle stappen correct zijn gevolgd dan kom je ju een goed werkent robotje uit volgens de foto.
