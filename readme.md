# LineFollower

lege repository die je als template kan gebruiken om een eigen repository te starten voor uw linefollower project

![1000008286](https://github.com/user-attachments/assets/13b22dbe-1b83-418a-956b-3eefd22c9617)
  
## specifications

microcontroller: ATmega32u4

motors: MiniDCN20,6V-30rpm

h-bridge: DRV8833 Dual Motor Driver Carrier

sensors: QTR-8A

batteries:  18650 Li-ion batterijen

wireless communication: HC-05 

distance sensor - motors: 95mm

weight: 211g

speed: 0.6m/s
